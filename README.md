## 🤝 Contributing

Contributions are **always welcome!** If you know a free tool or resource that belongs here:

1. **Fork** this repository
2. **Create** a new branch: `git checkout -b add/your-resource-name`
3. **Add** your resource to the correct category table
4. **Commit** your change: `git commit -m "Add: [Resource Name] to [Category]"`
5. **Push** your branch: `git push origin add/your-resource-name`
6. **Open** a Pull Request — describe what you added and why it's useful ✅

> **Guidelines:** Only free resources. No referral links. Keep descriptions concise (1 line). Verified working links only.
